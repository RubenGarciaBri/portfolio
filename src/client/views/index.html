Readable

Readable is a modern social media, online community discussion app.
This is my flagship project which incorporates elements similar to other famous social platforms such as Reddit and Facebook.

You can add posts to the main feed that anyone can interact with. Some of the cool features are: Upvotes and downvotes, add to favourites, comments, a live search bar and notifications. You even have your own public profile that you can customise.

This app uses Firebase Cloud Functions, the Firestore Database, Storage and Authentication on the back end, which means it's full-stack. You can create your own account, log-in and all the changes you make will be saved in the cloud in real time.

Would You Rather

This is an online version of the classic "Would you rather" game.
In order to play you can either create your own polls or answer any of the already open ones.

When you complete a poll, the app will show you what the other players answered and it will be moved to the "Answered" category. Every time you either create or complete a poll, you will receive 1 point that will be reflected in the leaderboard page.

The app lets you select from 5 out-of-the-box avatars that you can switch between to see the results in real time. Local storage is also used so all your progress is saved every time you come back.

Contentful Blog

This project is a fictional personal blog that uses the static site generator Gatsby and Contenful as a headless CMS.

This site uses GraphQL queries to fetch all the published blog posts on the Contenful dashboard. It also automatically deploys to Netlify every time a new blog post is published, deleted or edited which makes it a very easy-to-use solution for people with little programming knowledge.

The motivation behind this project was to learn modern blogging methods that use headless content management systems and static site generators, as opossed to other tradional technologies such as Wordpress.

Flashcards

This is a study app that helps you memorise information. You can create cards with questions and practice recalling the answers. You can also categorise all your cards into decks for better organisation.

Whenever you want to practice, all you have to do is click on "Start Quiz" and then select whether you got the question right or wrong. The app will also indicate the remaining cards left on that deck.

By the end, you will be able to see the results, including a total percentage so you can assess how well you know your stuff.
